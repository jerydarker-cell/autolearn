# AutoLearn v15.8.2 Public Safe Release Checklist

## Trước khi public
- [ ] App mở được trên Streamlit Cloud sau Reboot.
- [ ] App Health Check không có lỗi đỏ quan trọng.
- [ ] Public Safe Center: bộ 600 câu đủ 600/600.
- [ ] Đáp án khớp phương án.
- [ ] Đã kiểm tra trạng thái 60 câu điểm liệt.
- [ ] Terms & Privacy hiển thị rõ.
- [ ] Admin Mini Dashboard mở được.
- [ ] Test tài khoản A/B không trộn dữ liệu.
- [ ] Google Veo/TikTok chỉ load khi mở trang riêng.
- [ ] Mobile test trên điện thoại thật đạt.

## Sau khi public
- [ ] Ghi lỗi thực tế vào Deploy Hotfix Guide/Public Safe Center.
- [ ] Sửa theo lỗi thật, không nhồi thêm tính năng lớn.
