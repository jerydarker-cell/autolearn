# AutoLearn v15.7 Official Push Mobile

## Nâng cấp chính
- Thêm mục **Bộ đề chính thức 2025** theo cấu trúc bộ 600 câu CSGT áp dụng từ 01/06/2025.
- Có import CSV để đưa trọn bộ đề chính thức vào app theo từng hạng bằng.
- Thêm Web Push/PWA notification: service worker, manifest, VAPID, script gửi push.
- Thêm Mobile Pro Mode: giao diện học trên điện thoại rõ hơn, nút lớn hơn, flow học mỗi ngày.

## Lưu ý về bộ đề chính thức
Gói này có khung chuẩn và importer. Để có trọn 600 câu chính thức, hãy tải file từ CSGT/Bộ Công an và chuyển sang CSV theo template trong app. Không nên dùng bộ đề không rõ nguồn khi ôn thi thật.

## Deploy
Upload toàn bộ thư mục lên GitHub, không upload riêng app.py. Repo cần có app.py, modules/, static/, scripts/, sql/, requirements.txt, packages.txt, .streamlit/.
